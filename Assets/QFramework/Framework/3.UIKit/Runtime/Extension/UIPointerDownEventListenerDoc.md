# UIPointerDownEventListener

# 继承关系

继承自：MonoBehaviour

实现接口：IPointerDownHandler

# 描述

> 挂载组件
>
> 监听点击按下事件

# 公共属性

|                        |          |
| ---------------------- | -------- |
| **OnPointerDownEvent** | 点击事件 |

# 静态方法

|                         |                                   |
| ----------------------- | --------------------------------- |
| **Get**                 | 执行 **CheckAndAddListener** 方法 |
| **CheckAndAddListener** | 检查并添加此组件                  |

# 实现接口

|                   |                             |
| ----------------- | --------------------------- |
| **OnPointerDown** | 执行 **OnPointerDownEvent** |
